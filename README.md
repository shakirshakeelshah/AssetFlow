# Finance Dashboard

A modern full-stack personal finance tracking application with user authentication, built with React and Node.js.

## ✨ Features

- User registration and secure login (JWT)
- Add, view, and delete financial transactions
- Real-time dashboard with summary cards (Balance, Income, Expenses)
- Spending insights and analytics
- Balance trend and spending breakdown visualizations
- Dark/Light mode toggle with persistence
- Fully responsive design (works great on mobile and desktop)
- Backend API with data persistence

## 🛠 Tech Stack

**Frontend:**
- React + TypeScript
- Tailwind CSS
- Vite
- Chart.js + react-chartjs-2
- Axios

**Backend:**
- Node.js + Express
- JWT Authentication
- bcryptjs (for password hashing)
- lowdb (lightweight JSON database)
